# gzhu0610.github.io

Personal portfolio site for Guanyu Zhu.

Files:
- `index.html` contains the site structure and copy.
- `styles.css` contains the visual design and responsive layout.
- `script.js` contains scroll reveal and active-section behavior.

To preview locally, run `python3 -m http.server` in this directory and open the site in a browser.
